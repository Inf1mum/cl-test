# CL Test

Task - https://docs.google.com/document/d/1vTzjGuXnjUQ-YAaNMQZe9HkXpjI-VGM7RgKOuqtFr5E/edit

### Prerequisites

* JDK 8
* Maven v3.8.1+
* MySQL v8+


### How to run:

```
mvn spring-boot:run
```

### Run MySQL in docker
```
docker-compose up -d
```
DB will be accessible with:
  - url - `jdbc:mysql://localhost:3306/cl`
  - user - `root`
  - password - `admin`
